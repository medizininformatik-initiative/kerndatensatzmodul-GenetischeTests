### Externe ValueSets

Die folgenden ValueSets werden aus dem [HL7 Genomics Reporting Implementation Guide](http://hl7.org/fhir/uv/genomics-reporting/STU3/) übernommen:

| ValueSet | Beschreibung | Canonical URL |
|----------|--------------|---------------|
| HGNC ValueSet | Gen-Symbole und -Namen | `http://hl7.org/fhir/uv/genomics-reporting/ValueSet/hgnc-vs` |
| HGVS ValueSet | Sequenzvarianten-Notation | `http://hl7.org/fhir/uv/genomics-reporting/ValueSet/hgvs-vs` |
| DNA Change Type | DNA-Veränderungstypen | `http://loinc.org/vs/LL379-9` |
| Molecular Consequence | Molekulare Konsequenzen | `http://hl7.org/fhir/uv/genomics-reporting/ValueSet/molecular-consequence-vs` |
| Functional Effect | Funktionelle Effekte | `http://hl7.org/fhir/uv/genomics-reporting/ValueSet/functional-effect-vs` |
| Clinical Significance | Klinische Signifikanz | `http://loinc.org/vs/LL4034-6` |
| Evidence Level | Evidenzlevel | `http://hl7.org/fhir/uv/genomics-reporting/ValueSet/evidence-level-example-vs` |
| Therapeutic Implication | Therapeutische Implikationen | `http://hl7.org/fhir/uv/genomics-reporting/ValueSet/therapeutic-implication-vs` |
| Condition Inheritance | Vererbungsmuster | `http://hl7.org/fhir/uv/genomics-reporting/ValueSet/condition-inheritance-vs` |
| Variant Confidence Status | Varianten-Konfidenzstatus | `http://hl7.org/fhir/uv/genomics-reporting/ValueSet/variant-confidence-status-vs` |

Weitere Details zu den einzelnen ValueSets finden Sie in den entsprechenden Unterseiten.
